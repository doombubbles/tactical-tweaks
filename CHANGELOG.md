# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

## [1.2.2] - 2026-06-24

- Fixed Pursuit Path Prioritization overriding the buttons for Lock in Place and Patrol Points

## [1.2.1] - 2026-06-07

- Updated for BTD6 v55
- Fixed Brickell Mermonkeys

## [1.2.0] - 2026-05-26

- Added Pursuit Path Prioritization (default on)
  - Allows Heli Pilots to choose to prioritize Bloons on specific paths.
- Added Hero Boost Debt (default on)
  - Replaces the Monkey Money Hero Boost with a button that lets you purchase the next level up of the hero by going into debt.

## [1.1.7] - 2026-05-11

- Fixed ice monkey paragon console error when using ability choice
- Added a tweak for making Obyn's Druid buffs be global

## [1.1.6] - 2026-04-12

- Fixed some tweaks for v54

## [1.1.5] - 2026-02-22

- Fixed some tweaks for v53

## [1.1.4] - 2026-01-12

- Fixed Better Ezili Totem not aligning with its toggled setting state
- Fixed a scaling issue with Sticky Bomb Attack Speed for very highly buffed attack rate
- Added a temporary fix for the Ice Monkey Paragon ability not activating its full slow if a round ends during it (this should get fixed for real by NK soon)

## [1.1.3] - 2025-12-11

- Fixed Buffable Paragons not immediately getting the pre-existing zone buffs they're within on creation

## [1.1.2] - 2025-12-03

- Fixed Brickell Mermonkeys for v52
- Fixed an error message that could appear in the log about Sticky Bomb Attack Speed
- Made the Overclock Cash tweak also affect cash generating abilities

## [1.1.1] - 2025-11-06

- Fixed the default value of Buffable Paragons

## [1.1.0] - 2025-11-06

### New Tweaks

- **Buffable Paragons** (Default Off)
  - Makes Paragons able to receive buffs like normal towers.
- Brickell Mermonkeys (Default On)
  - Makes Brickell's effects apply to non-exclusively water based towers again.
- Elite Defender Target Switch (Default Off)
  - An Elite Defender will switch to First targeting after a Bloon leak activates its ability.
- Sticky Bomb Attack Speed (Default Off)
  - Makes attack speed buffs cause Sticky Bombs to blow up quicker. (Also affects Master Bomber)
- Better Laser Blasts (Default Off)
  - Makes the Laser Blasts upgrade give a minor Attack Speed buff for Super Monkeys. Plasma Blasts unaffected.
  - Rates for 0/0/0, 1/0/0, 2/0/0 are .045, 0.45, .03 -> .045, .04, .03
- Better Trojan (Default Off)
  - Makes Benjamin's Bloon Trojan use the real internal layer count for Bloons so that it no longer loses money.
- Etienne Targeting (Default Off)
  - Replaces Etienne's targeting options with the standard Fist, Last, Close, Strong.
- Beast Handler Lone T5s (Default Off)
  - Allows Tier 5 Beast Handlers to be purchased even without any other Beast Handlers contributing to them.
  - Megalodon still requires 84 Beast Power to insta kill BAD Bloons.

## [1.0.1] - 2025-08-27

- Fix for v50
- New toggleable utilities for opening up some additional interesting farming strategies
  - Overclock Cash - Makes the Overclock buff increase end of round cash generated by towers.
  - Better Monkey City - Makes the Monkey City income buff affect more things, including Lead to Gold, IMF Loan, Monkey-Nomics, Monkey Pirates, and similar effects.
  - Better Central Market - Makes the Central Market buff affect all non-Support towers' income generation not just Merchantmen.

## [1.0.0] - 2024-10-25

- Initial Release
